# Templates de Follow-Up — EZContact

Scripts listos para usar cuando un prospecto respondió o cuando hay que reactivar.

---

## 📬 1. Prospecto dijo "sí, cuándo pueden llamar" (sin confirmar horario)

**Usar cuando:** VETME, otros prospectos que piden llamada pero no confirman fecha

**Email de seguimiento:**
```
Asunto: Re: [tema original]

Hola [Nombre], buen día.

Solo quería confirmar si [jueves/viernes] a las [hora] funciona para la llamada.

Son 15 minutos. Si ese horario no funciona, me dices cuándo y nos adaptamos.

Saludos,
Victor
```

**WhatsApp (más directo):**
```
Hola [Nombre], soy Victor de EZContact.
Les escribí por email para agendar la llamada.
¿Funciona hoy a las 2pm o 4pm?
Solo 15 minutos.
```

## 📬 2. Recordatorio de reunión confirmada (día anterior)

**Usar cuando:** Idiomas CUC, cualquier reunión confirmada por Zoom

**Email (día anterior):**
```
Asunto: Re: [tema] — Recordatorio reunión mañana 10am

Hola [Nombre],

Solo un recordatorio rápido de nuestra reunión mañana [fecha] a las 10am.

Link de Zoom: https://calendly.com/ezcontact/demo

Si necesitas cambiar el horario, avísame con tiempo.

¡Hasta mañana!
Victor
```

**WhatsApp (mañana del día):**
```
Buenos días [Nombre] 🙌
Confirmando nuestra llamada de hoy a las [hora].
¿Todo bien de su lado?
```

## 📬 3. Prospecto no respondió (2-3 días después del primer outreach)

**Usar cuando:** No hay respuesta al email inicial

```
Asunto: Re: [tema original]

Hola [Nombre],

Solo revisando si mi email anterior llegó bien.

En resumen: tenemos una solución de IA para WhatsApp que puede [beneficio específico del vertical].

¿Vale la pena 15 minutos para revisarlo?

Si no es el momento, no hay problema — solo díganme.

Saludos,
Victor
```

## 📬 4. Prospecto con sistema anti-spam (verificación de email)

**Usar cuando:** Recibimos email de verificación de contacto@laclinicadental.org y similares

**Opción A — Responder la verificación:**
Hacer clic en el link del email de verificación O responder el email dejando el asunto intacto.

**Opción B — Intentar otro canal:**
```
Buscar en Google: "[empresa] teléfono" o "[empresa] WhatsApp"
→ Llamar directamente
→ WhatsApp si aparece número
```

**Opción C — LinkedIn:**
```
Buscar la empresa en LinkedIn
→ Conectar con la persona de administración o dueño
→ Mensaje directo
```

## 📬 5. Prospecto interesado pero "ahora no" (timing issue)

```
Asunto: Re: EZContact — En 30 días

Hola [Nombre],

Entendido, el timing importa.

Te dejo una pregunta: ¿qué tendría que pasar en su negocio para que esto se volviera prioridad?

No es para presionar — es para entender si tiene sentido volver a hablar después.

Saludos,
Victor
```

## 🎯 Reglas de Oro — Follow-Up

| Situación | Acción | Timing |
|-----------|--------|--------|
| Respondió positivo | Confirmar fecha inmediatamente | Mismo día |
| Confirmó reunión | Recordatorio el día anterior | D-1 |
| No respondió email | Seguimiento 1 | D+3 |
| No respondió follow-up 1 | Seguimiento 2 (diferente canal) | D+7 |
| Respondió "ahora no" | Seguimiento suave | D+30 |
| Respondió "no gracias" | Archivar | - |

**Máximo 3 toques** antes de archivar un lead.
**Siempre dar opción de salida** — "si no es el momento, no hay problema".

---

## 📱 WhatsApp vs Email

**Usar WhatsApp cuando:**
- Ya hubo intercambio previo por email
- La reunión es hoy o mañana (urgente)
- El lead es muy cálido y el email no tuvo respuesta

**Usar Email cuando:**
- Primer contacto o sin respuesta aún
- Adjuntar materiales o propuesta
- Documentar el proceso para CRM
